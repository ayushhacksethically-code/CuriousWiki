📖 Arch Linux Bible

June 30, 2026 5 min read linux, arch, setup

Arch Linux एक lightweight, customizable, और user-centric Linux distribution है। यह guide, Arch Linux के अंदर system configuration update करने, server status check करने, और dependencies manage करने के लिए एक comprehensive reference (bible) के रूप में काम करती है।

1. Pacman Package Manager

System databases को sync करने, packages search करने, और installs manage करने के लिए primary tools:

# System synchronization aur updates pull karein (full upgrade)
sudo pacman -Syu

# New package install karein
sudo pacman -S package_name

# Package aur uske unnecessary dependencies remove karein
sudo pacman -Rns package_name

# Local package database search karein
pacman -Ss query_word

# Local package cache complete clear karein
sudo pacman -Scc

2. Systemd Services Management

Arch init boot system processes (services) को control और configure करना:

# service status check karein
systemctl status service_name

# service start karein (current session)
sudo systemctl start service_name

# service boot startup par enable karein
sudo systemctl enable service_name

# service check default run status
systemctl is-active service_name
Important: AUR Safety check

AUR helper tools (जैसे yay or paru) का compile/install script run करने से पहले हमेशा PKGBUILD script file को manually audit कर लें।

3. AUR (Arch User Repository)

Community driven custom programs compile करना और install करना:

# aur sources download karein
git clone https://aur.archlinux.org/yay-bin.git

# package directory me jump karein
cd yay-bin

# compile aur local install complete karein
makepkg -si

4. System Logs (Journald)

Core kernel logs और program runtime errors monitor करना:

# real-time live logs tail show karein
journalctl -f

# current boot process debug logs extract karein
journalctl -b

# specific service runtime error code list search karein
journalctl -u service_name --since "1 hour ago"
