📝 Terminal Text Editors & Setup Guide

July 01, 2026 5 min read editors, config, linux

Linux Terminal में direct config files और code edit करने के लिए multiple options उपलब्ध हैं। इस guide में हम प्रमुख editors और default configurations के बारे में जानेंगे।

1. Core Terminal Editors (मुख्य टेक्स्ट एडिटर्स)

⚫ Nano Editor

यह beginners के लिए सबसे सरल Editor है। इसके shortcuts (जैसे copy, save, exit) screen के नीचे हमेशा visible रहते हैं।

  • Save File: Ctrl + O
  • Exit: Ctrl + X
  • Search Text: Ctrl + W

⚫ Vim / Vi Editor

Vi (Visual Instrument) पारंपरिक Unix editor है। Vim (Vi Improved) उसका extended standard version है। यह keyboard-driven modes पर काम करता है:

  • Normal Mode: File navigation के लिए (Default mode).
  • Insert Mode: Text लिखने के लिए (i दबाएं)।
  • Command Mode: Save/Exit करने के लिए (Esc के बाद : दबाएं)। :wq (save + exit) या :q! (discard + exit)।

⚫ GNU Emacs

यह एक extremely extensible, screen-oriented text editor है। इसे "an editor within an OS" भी कहा जाता है क्योंकि इसमें builtin terminal console, file manager, email client, और macros integrations मिलते हैं। Lisp language पर configure होने के कारण यह infinite levels of customizations support करता है।

  • Save File: Ctrl + X, फिर Ctrl + S
  • Exit: Ctrl + X, फिर Ctrl + C

2. Distro-specific Install Commands

अलग-अलग Linux distribution configurations में इन editors को install करने के commands:

# --- 1. Debian/Ubuntu Based Systems ---
sudo apt update
sudo apt install nano vim emacs

# --- 2. Fedora/RHEL Based Systems ---
sudo dnf install nano vim-enhanced emacs

# --- 3. Arch Linux Based Systems ---
sudo pacman -Syu nano vim emacs-nox

3. Setting Default System Editor (डिफ़ॉल्ट एडिटर सेट करना)

जब आप git commit या visudo run करते हैं, तो OS automatically एक default editor launch करता है। इसे configuration paths में set करने के multiple ways हैं:

Method A: Shell Environment Variables (Universal Method)

अपनी profile config (जैसे ~/.bashrc या ~/.bash_profile) में Editor variable export करें:

# Default editor को Vim सेट करने के लिए:
export EDITOR="vim"
export VISUAL="vim"

# Default editor को Nano सेट करने के लिए:
export EDITOR="nano"
export VISUAL="nano"

# Save करने के बाद profile reload करें:
source ~/.bashrc

Method B: System-wide Alternatives (Debian/Ubuntu)

Debian-based OS में system-wide global interactive shell selection utility का use करें:

sudo update-alternatives --config editor
# Menu में से अपने मनपसंद editor का sequence number enter करें
