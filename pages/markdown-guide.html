📝 Markdown & Callouts Cheatsheet

June 30, 2026 3 min read formatting, reference

यह cheatsheet CuriousWiki के अंदर styled सभी standard formatting components के visual और code examples प्रदान करती है। text, code blocks, lists, warnings, या tables को style करते समय इस page को refer करें।

1. Typography & Headers

Headings, documents के flow को structure करने में मदद करती हैं। हम H1, H2, H3, और H4 sizes को support करते हैं:

H1 Document Header (accent bottom-border के साथ)

H2 Major Section Header

H3 Subsection Header

H4 Small Subsection Title

Inline text emphasis को style करने के लिए, आप bold text (priority के लिए) और italicized text (emphasis के लिए) का उपयोग कर सकते हैं। standard paragraphs, effortless long-form reading के लिए आरामदायक line-spacing (1.65) के साथ set हैं।

2. Styled Callout Blocks

Callouts, info, warnings, success reports, और critical errors के लिए बेहतरीन हैं। इन्हें modern colored border accents और matching icons के साथ render किया गया है:

Info Box

उपयोगी tips, सहायक guidance, background context, या छोटे references यहाँ जाते हैं।

Warning Alert

Cautions, संभावित errors, deprecations, या मुद्दे जिन पर ध्यान देने की आवश्यकता है।

Success Panel

सकारात्मक completions, verified setups, सही choices, या resolved logs.

Danger Alert

उच्च जोखिम वाले (high-risk) instructions, security alerts, breaking changes, या terminal failure indicators।

3. Lists & Nested Formatting

आप styled ordered lists, bullet point lists, और nested hierarchies बना सकते हैं:

  • Unordered bullet item 1
  • Unordered bullet item 2
    • Nested bullet item 2a
    • Nested bullet item 2b
  • Unordered bullet item 3
  1. Step number 1
  2. Step number 2
    1. Substep number 2a
    2. Substep number 2b
  3. Step number 3

4. Keyboard Shortcut Elements

Keyboard inputs को display करने के लिए, tactile physical buttons render करने के लिए <kbd> element का उपयोग करें। उदाहरण के लिए:

Copy करने के लिए Ctrl + C दबाएं, paste करने के लिए Ctrl + V दबाएं, या active dialog boxes को बंद करने के लिए Esc दबाएं।

5. Data Tables

Tables को automatic रूप से highlighted header rows, thin border dividers, और high-contrast text के साथ style किया जाता है:

Variable Type Default Description
theme String "dark" Active CSS visual skin (dark/light) निर्दिष्ट करता है।
sidebarWidth Integer 280px Sidebar drawer की responsive width.
isSubpage Boolean false Static assets के लिए path depth calculations निर्धारित करता है।

6. Preformatted Code Blocks

Code snippets, standard syntax frames में लपेटे गए हैं जिनमें language tags और Copy buttons होते हैं:

/* Custom styled card variables */
.card-item {
    background-color: var(--bg-secondary);
    border: 1px solid var(--border-color);
    border-radius: 8px;
    padding: 1.25rem;
    transition: all 0.25s ease;
}
