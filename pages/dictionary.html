📖 Jargon Dictionary

June 30, 2026 Glossary vocabulary, references

इस wiki में उपयोग किए जाने वाले technical jargon, programming concepts, और command-line terms का एक central glossary (शब्दकोश)। नए शब्दों को सीखने और cross-reference करने के लिए इसका उपयोग करें।

A C D F G L M N P R S V
A
  • API (Application Programming Interface)
    अलग-अलग software applications/components के बीच data exchange और communication स्थापित करने का standard माध्यम।
  • Async (Asynchronous)
    वह execution flow जिसमें task background में run होता है और main thread (UI loop) को block नहीं करता, जिससे user interface lag-free रहता है।
  • AUR (Arch User Repository)
    Arch Linux users के लिए एक community-driven package repository, जहाँ user-built Custom packages की recipes (PKGBUILD) store होती हैं।
C
  • Callout
    Document layout में एक distinct styled container block, जो specific colors और dynamic icons के साथ important warnings, notes, or success logs को highlight करता है।
  • CLI (Command Line Interface)
    Terminal console पर typing commands के द्वारा user interactions और computer file system operations को perform करने का interface.
  • Commit
    Version control systems (जैसे Git) में codebase changes का snapshot local history में snapshot-id के साथ save करना।
D
  • DOM (Document Object Model)
    HTML nodes का tree structure structure, जिसे JavaScript scripts runtime पर dynamically query, alter, style, या delete कर सकते हैं।
F
  • Fuzzy Search
    Exact key matching के बजाय logical approximations, spelling mistakes tolerance, और prefix query inputs के आधार पर accurate matching results ढूँढने की method.
G
  • Git
    एक open-source, distributed version control application जो source repositories में file changes, revisions, और branches merging timeline को track करती है।
L
  • LocalStorage
    Web browsers का key-value pair database engine, जो dynamic site properties (जैसे UI theme state) को session/tabs resets के बाद भी permanent save रखता है।
M
  • MiniSearch
    Client-side execution structures के लिए optimized, pure JavaScript full-text indexing engine library, जो tag query processing और fuzzy searches support करता है।
N
  • Nim
    एक statically typed programming language, जो directly zero-overhead compiled C/C++ native code output produce करती है और super-lightweight servers execute करती है।
P
  • Pacman
    Arch Linux OS का package management CLI tool, जो database dependencies sync करने और system applications upgrades compile करने में use होता है।
  • Promise
    JavaScript engine का object जो network fetch operations या execution wait timers के completion value (success or failure state) को represent करता है।
R
  • Rebase
    Git repository workflow model, जहाँ commits history timeline clean alignment structure maintain करने के लिए feature commits base sequence reset की जाती है।
S
  • Static Site
    HTML, CSS, और JS components से बनी website, जो servers par bina databases runtime computation run किए pre-built state assets direct render करती है।
  • Systemd
    Linux distributions init processes system component daemon, जो boot sequence bootstrap initialization processes control करता है और applications daemons state start/stop करता है।
V
  • Variables
    Software programs memory containers, जो variable names के pointers structure के dynamically changing values, flags, array records list preserve रखते हैं।
