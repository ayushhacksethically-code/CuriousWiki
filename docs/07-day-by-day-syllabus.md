# 📚 07. 180-Day Systems Engineer Mastery Path (डे-बाई-डे मास्टर क्लास)

यह 6 महीने (180 दिन) का एक विस्तृत दैनिक कार्यक्रम (day-by-day plan) है. इसे follow करके आप basic level से coding, CLI shells, GNU/Linux systems, HTML5 UI, compiler design, databases, और security को master कर सकते हैं. 

हर दिन के लिए **Topic**, **Daily Challenge (व्यावहारिक कार्य)**, और **Mastery Check (यह जानने के लिए कि आप इसे सीख चुके हैं)** दिए गए हैं.

---

## 📁 180-Day Overview (6 महीने का सारांश)
* **Month 1 (Days 001 - 030)**: GNU/Linux System Admin & Bash CLI Mastery (लिनक्स एडमिनिस्ट्रेशन और बैश शेल)
* **Month 2 (Days 031 - 060)**: PowerShell Core (pwsh) Object Scripting (पॉवरशेल कोर)
* **Month 3 (Days 061 - 090)**: Systems Foundations: C Programming & Memory Management (सी लैंग्वेज और मेमोरी)
* **Month 4 (Days 091 - 120)**: Python Automation & SQLite Databases (पायथन और डेटाबेस)
* **Month 5 (Days 121 - 150)**: Web Engineering: HTML5, CSS3, JS & Browser WASM (वेब इंजीनियरिंग)
* **Month 6 (Days 151 - 180)**: Nim Systems Programming, Cybersecurity & Capstone (निम और सुरक्षा)
