# UNIX & Linux History Timeline

यहाँ UNIX and Linux ke evolution (इतिहास) की complete chronological checklist/timeline दी गई है।

| Year | Developments (विकास और घटनाएँ) |
| :--- | :--- |
| **1965** | GE computer ke liye ek naya **operating system** banane ke liye AT&T, MIT, aur GE ke **joint venture** ke roop mein **Multics project** ki shuruat hui. |
| **1969** | AT&T Bell Labs ke researchers Ken Thompson, Dennis Ritchie, J. F. Ossanna, aur R. H. Canaday ne MULTICS ke alternative ke roop mein ek prototype **file management system** banaya. Us samay ke commercial systems puri tarah se **assembly language** mein likhe jate the. UNIX ke goals mein se ek goal yeh tha ki iska ek chhota **kernel** ho jise **assembler** mein likha jaye, aur baaki hissa kisi bhi **high-level language** mein ho. Unix mein ek **hierarchical file system** aur **utility programs** ka collection bhi tha. |
| **1970** | Brian Kernighan ne UNICS (UNIplexed Information and Computing System) naam diya. DEC (Digital Equipment Corporation) PDP-11 (ek state-of-the-art $65,000 computer jisme 24 kilobytes **RAM** aur 512 kilobytes **disk space** tha) ke aane ke baad Unix ka development aur tezi se badha. Thompson ne FORTRAN aur BCPL ke alternative ke roop mein **B programming language** develop ki. |
| **1971** | 1st Unix version, V1, ka upyog sirf Bell Labs ke andar hi kiya gaya. Development cost ko justify karne ke liye, Bell Labs ke patent department mein UNIX ka upyog (**assembly-language** mein coded **troff** ke sath) pehle **word-processing programs** mein se ek ke roop mein kiya gaya. **B language** ko behtar banaya gaya aur uski successor language ka naam **C** rakha gaya. |
| **1972** | M. D. McIlroy ne 'pipes' ka ek bilkul naya idea introduce kiya. June mein Version 2 aya, jiske 10 Unix **installations** the. |
| **1973** | Version 3 ke 16 Unix **installations** hue. November mein Version 4 ko **C language** mein rewrite kiya gaya, jisse Unix ki **portability** bahut aasan ho gayi. |
| **1974** | S. R. Bourne ne **Bourne Shell** (`/bin/sh`, jise '$' se indicate kiya jata hai) develop kiya. June mein Version 5 aya, jiske lagbhag 50 Unix **installations** ka anuman tha. |
| **1975** | AT&T ne Version 6 ko universities ke liye low cost par lease par diya, jisse UNIX ka upyog widespread (bohot zyada) ho gaya. Thompson ne UC Berkeley mein ek saal bitaya aur Unix ke ek **BSD variant** ke development ko lead kiya. UC Berkeley ke graduate student Bill Joy (jinhone baad mein Sun Microsystems shuru kiya) ne **C-shell** (`/bin/csh`, jise '%' se indicate kiya jata hai) aur **vi text editor** develop kiya. TENEX-style **C-shell** develop kiya gaya (`/bin/tcsh`). AT&T ke David Korn ne **Korn shell** (`/bin/ksh`) develop kiya. |
| **1976** | **Emacs text editor** ko originally Richard Stallman dwara likha gaya. |
| **1977** | **1BSD** release hua. Tom Duff aur Byron Rakitzis ne **rc shell** develop kiya. |
| **1978** | UC Berkeley ke students, jinhe "Berkeley Software Distribution" ke naam se jana jata tha, unhone UNIX ka apna khud ka variant develop kiya jise **BSD** kaha gaya. **2BSD** release hua aur iski 75 copies distribute ki gayin. Duniya bhar mein 600 Unix **installations** ho chuke the. |
| **1979** | Private companies ne Unix ke **commercial versions** ko port karna shuru kar diya. BSD ne **3BSD** release kiya. AT&T ne 40KB-**kernel** wala **Version 7** release kiya. |
| **1980** | Microsoft ne **Xenix** release kiya, jo Unix ko desktop computers par lane ki pehli koshish thi. October mein BSD ne **4.0 BSD** release kiya. |
| **1982** | AT&T ne Unix ka apna pehla **commercial version** release kiya, jise **System III** kaha gaya. 1982 mein **Ksh (Korn shell)** ko AT&T Bell Labs mein working condition mein deliver kiya gaya. |
| **1983** | Computer Research Group (CRG), UNIX System Group (USG), aur Programmer's WorkBench (PWB) ka merge hua aur unhone milkar **UNIX System Development Lab** banayi. AT&T ne **System V** release kiya, jisme Xenix aur dusre variants ko incorporate kiya gaya tha. BSD ne **4.2BSD** release kiya jisme **TCP/IP networking protocols** ka complete implementation shamil tha, jisme **telnet** aur **ftp** bhi the. UNIX flavors ko jitna ho sake standardize karne ke koshish mein **SVID (System 5 Interface Definition)** release kiya gaya. |
| **1984** | Duniya bhar mein lagbhag 100,000 UNIX **installations** ka anuman tha. U.S. government ne AT&T par **monopolistic practices** (ekadhikaar) ke charges lagaye aur AT&T ko apne interests ko **divest** karne ke liye force kiya gaya. AT&T ne **SVR2** release kiya, jisme 4.2BSD ke kai features ko incorporate kiya gaya tha. Vendors ke **X/Open consortium** ki sthapna hui (jise baad mein **The Open Group** ke naam se jana gaya) jise UNIX ka **trademark** mila. Richard Stallman ne ek free UNIX clone ke roop mein **GNU (GNU's Not UNIX)** develop kiya. |
| **1985** | February mein AT&T ne **Version 8** release kiya. Paul Falstad ne **zsh** (Z shell) develop kiya. |
| **1986** | September mein AT&T ne **Version 9** release kiya. DEC, jo VAX/VMS ko support kar raha tha, use UNIX ko ek inexpensive alternative ke roop mein acknowledge aur support karne ke liye force hona pada. **Rc shell** ko **es** shell mein upgrade kiya gaya. |
| **1987** | Duniya bhar mein lagbhag 100,000 Unix **installations** ka anuman tha. |
| **1988** | **Unix International (UI)** aur **Open Software Foundation (OSF)** ka gathan kiya gaya. System V, BSD, aur SunOS ke combo ke roop mein **SVR4** release kiya gaya. |
| **1989** | October mein AT&T ne final version **V10** release kiya. Ksh ke free alternative ki chah mein, GNU advocates ne **bash (Bourne-again shell)** develop kiya. |
| **1991** | Unix Systems Laboratory (USL) ko ek alag company ke roop mein spin off kiya gaya, jisme majority ownership AT&T ki thi. OSF ne **OSF/1** release kiya. Linus Torvalds ne **Linux kernel** (Linus ka Minix, jise 'lin-ux' pronounce kiya jata hai) release kiya. |
| **1992** | <ul><li>USL (Unix Systems Laboratories) ne UNIX **trademark** aur **copyright infringements** ko lekar BSDI (Berkeley Software Design Inc.) ke khilaf ek **lawsuit** file kiya.</li><li>Linus Torvalds ne Linux **kernel** ko **GNU General Public License (GPL)** ke tehat **re-license** kiya, jisse yeh sach mein free software ban gaya.</li><li>Intel 386 processors ke liye ek free BSD port ke roop mein **386BSD** release kiya gaya.</li></ul> |
| **1993** | <ul><li>AT&T ne Unix Systems Laboratories (USL) ko Novell ko bech diya.</li><li>Novell ne UNIX **trademark** ko **X/Open consortium** (ab **The Open Group**) ko transfer kar diya.</li><li>Ian Murdock ne **Debian Linux distribution** banaya aur use launch kiya.</li><li>Patrick Volkerding dwara **Slackware Linux** release kiya gaya.</li><li>**FreeBSD** aur **NetBSD** version 1.0 officially release kiye gaye.</li></ul> |
| **1994** | <ul><li>Linus Torvalds ne officially Linux Version 1.0.0 release kiya.</li><li>**Red Hat Linux** ki sthapna hui aur isne apna pehla version release kiya.</li><li>USL vs. BSDI **lawsuit** settle ho gaya, jisse unrestricted BSD development ka rasta saaf ho gaya.</li></ul> |
